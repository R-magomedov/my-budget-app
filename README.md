# My Budget App

Учебный проект на React для учета доходов и расходов с возможностью добавления, удаления, поиска и фильтрации транзакций

## Функционал

- Просмотр списка транзакций
- Добавление и удаление транзакций
- Подсчет баланса, доходов и расходов
- Фильтрация по типу и поиск по названию
- Сохранение данных в localStorage
- Модальное окно с формой и валидацией

## Технологии

- React
- Context API
- SCSS Modules
- localStorage
- Vite

## Запуск

```bash
npm install
npm run dev
```
